# Study-react-koa

## Stack
```
# React
# Node (Koa)
# MongoDb
```

## node_module 설치

```
# server

$ cd/server
$ yarn install

# frontend

$ cd/frontend
$ yarn install
```

## 개발모드 자동 실행

```
# server

$ cd/server
$ yarn start [Listening to port 4000]

# frontend

$ cd/frontend
$ yarn start [http://localhost:3000/]
```

## 프로덕션 모드로 빌드
```
$ cd/frontend
$ yarn build
```
